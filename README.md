## Problem

Sales teams miss the highest-intent trigger event—job changes—because manual tracking is impossible at scale.

## Who Is This For?

B2B sales teams and recruiters

## Features

- **Silent LinkedIn profile monitoring**
- **Real-time job change alerts via Slack/email**
- **Contact enrichment with new work emails**
- **CRM auto-update integration**
- **Team collaboration on hot leads**

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/malikmuhammadsaadshafiq-dev/mvp-jobchangealert.git
cd mvp-jobchangealert
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
